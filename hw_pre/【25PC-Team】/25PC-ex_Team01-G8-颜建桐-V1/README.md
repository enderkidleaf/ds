## 1. 项目简介
本项目通过对当当网 Python 相关图书的数据抓取、清洗与深度分析，揭示当前编程图书市场的出版趋势、价格策略及读者偏好。项目涵盖了完整的自动化数据流水线：从原始网页爬取到结构化数据处理，最后生成多维度的可视化分析报告。

## 3. 项目核心阶段 (Pipeline)

### **第一阶段：数据采集**
* **脚本**: `01_run_scraper.ipynb`
* **功能**: 自动化爬取当当网搜索结果页，获取包含书名、作者、出版社、价格、评论数等原始信息。
* **产出**: `data_raw/python_books_raw.csv`。

### **第二阶段：数据处理**
* **脚本**: `02_data_clean.ipynb`
* **核心逻辑**: 进行文本标准化（去除营销词）、处理重复记录、识别并过滤异常值（如价格异常、年份错误）。
* **产出**: `data_clean/python_books_clean.csv`。

### **第三阶段：多维度分析**
* **脚本**: `03_data_analysis.ipynb`
* **工作内容**: 从时间（出版分布）、价格（折扣力度）、热度（评论统计）、出版社（市场占有率）等维度进行深度挖掘。
* **产出**: `output/` 文件夹下的系列可视化图表。

### **第四阶段：成果输出**
* **核心职责**: 基于分析目标完成全维度数据分析，并将分析结果与可视化图表进行整合。
* **产出**: 最终市场调研报告与完整的分析成果物。

## 4. 可视化成果展示 (Output)

我们通过以下 10 个维度对市场进行了深度解析：

* **01_time_distribution.png**: **年度出版分布** —— 展示 Python 图书在各年份的出版集中度。
* **02_year_heat.png**: **年份-热度交叉分析** —— 探索不同出版年份图书的市场关注度差异。
* **03_price_analysis.jpg**: **价格与折扣分布** —— 揭示市场定价策略与主流折扣水平。
* **04_category_price.png**: **分类价格分析** —— 比较不同技术领域（如基础入门 vs AI 进阶）的定价差异。
* **05_comments_analysis.jpg**: **评论数分布** —— 展现市场热度的“长尾效应”，识别爆款图书。
* **06_heat_level.png**: **市场热度分层** —— 对图书按受关注程度进行等级划分分析。
* **07_publisher.jpg**: **出版社竞争力分析** —— 统计各大出版社的市场份额与图书表现。
* **08_category.png**: **图书分类占比** —— 直观展示当前 Python 市场中各细分领域的流行程度。
* **09_comprehensive.jpg**: **综合指标分析** —— 结合年份、价格、热度的三维交叉洞察。
* **10_correlation.png**: **各项指标相关性** —— 揭示价格、折扣、评论数与年份之间的内在统计联系。
